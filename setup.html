云同步设置

配置 GitHub 实现多设备数据同步

第一步：获取 GitHub Token

  1. 1 访问 github.com/settings/tokens
  2. 2 点击 "Generate new token (classic)"
  3. 3 Note 填写：restaurant-data
  4. 4 勾选 "repo" 权限（全部 repo 权限）
  5. 5 点击 "Generate token" 并复制生成的 Token

第二步：配置 Token

Token 只会保存在您的浏览器本地，不会上传到任何地方
