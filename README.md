# 美味餐厅 - 扫码点餐系统

一个完整的餐饮扫码点餐系统，支持顾客扫码点餐、商家后台管理、历史订单查看和今日流水统计。

## 📁 项目结构

```
├── index.html          # 顾客点餐页面
├── admin.html          # 商家后台管理页面
├── qrcode.html         # 桌台二维码生成页面
├── css/
│   ├── style.css       # 主样式文件
│   └── admin.css       # 后台管理样式
├── js/
│   ├── data.js         # 数据模块（菜品、订单、本地存储）
│   ├── cart.js         # 购物车模块
│   ├── app.js          # 顾客端应用逻辑
│   └── admin.js        # 商家后台应用逻辑
└── images/             # 图片资源目录
```

## ✨ 主要功能

### 🍽️ 顾客端 (index.html)

- **扫码点餐**: 通过扫描桌台二维码进入点餐页面
- **菜品浏览**: 按分类浏览菜单（主食、荤菜、素菜、汤品、饮品、甜点）
- **热销推荐**: 显示热销菜品标签
- **购物车**: 
  - 添加/减少菜品数量
  - 实时显示总价
  - 底部购物车栏快速查看
- **订单提交**: 
  - 确认订单详情
  - 微信支付模拟
  - 下单成功提示
- **历史订单**: 查看历史订单记录

### 🏪 商家后台 (admin.html)

- **今日概览**:
  - 今日营收统计
  - 今日订单数量
  - 客单价计算
  - 待出餐订单数
  - 营收趋势图表
  - 热销菜品TOP5
  - 最近订单列表

- **订单管理**:
  - 查看所有订单
  - 按状态筛选（待确认、制作中、待出餐、已完成）
  - 按日期筛选
  - 更新订单状态
  - 查看订单详情

- **菜品管理**:
  - 添加新菜品
  - 编辑菜品信息
  - 上下架管理
  - 分类管理

- **营收统计**:
  - 今日/本周/本月营收汇总
  - 营收趋势图表
  - 详细营收明细

- **桌台管理**:
  - 查看所有桌台状态
  - 空闲/使用中状态显示
  - 生成桌台二维码

### 📱 二维码生成 (qrcode.html)

- 自动生成所有桌台的点餐二维码
- 支持单独打印某个桌台的二维码
- 打印友好的样式设计

## 🚀 使用方法

### 1. 直接打开使用

在浏览器中打开 `index.html` 即可体验顾客点餐页面。

### 2. 查看商家后台

打开 `admin.html` 进入商家管理后台。

### 3. 生成桌台二维码

打开 `qrcode.html` 生成所有桌台的二维码，用于打印。

### 4. 部署到服务器

将整个项目文件夹上传到任何支持静态文件的Web服务器即可。

## 💡 使用场景

1. **餐厅部署**:
   - 将 `qrcode.html` 生成的二维码打印出来
   - 放置在每个餐桌上
   - 顾客用微信扫码进入 `index.html` 页面点餐

2. **后厨管理**:
   - 在后厨电脑打开 `admin.html`
   - 实时查看新订单
   - 更新订单状态（制作中→待出餐→已完成）

3. **收银管理**:
   - 通过后台查看今日营收
   - 统计订单数据
   - 管理菜品信息

## 🛠️ 技术栈

- **前端**: HTML5, CSS3, JavaScript (原生)
- **图表**: Chart.js
- **图标**: Font Awesome
- **二维码**: QRCode.js
- **数据存储**: LocalStorage (本地存储)

## 📝 数据说明

### 菜品数据

菜品数据预置在 `js/data.js` 中，包含：
- 23个预设菜品
- 6个菜品分类
- 价格、描述、销量等信息

### 订单数据

- 首次打开会自动生成过去7天的模拟订单数据
- 新订单保存在浏览器本地存储中
- 数据仅保存在当前浏览器，清除缓存会丢失

## 🔧 自定义配置

### 修改桌台列表

在 `js/data.js` 中修改桌台数组：

```javascript
const tables = ['A01', 'A02', 'A03', 'A05', 'A06', 'A08', 'B01', 'B02', 'B03', 'B05', 'B06', 'B08'];
```

### 修改菜品数据

在 `js/data.js` 中的 `menuData` 数组中添加或修改菜品：

```javascript
{
    id: 24,
    name: "新菜品名称",
    category: "meat",  // 分类: staple/meat/vegetable/soup/drink/dessert
    price: 38,
    description: "菜品描述",
    image: "🍗",  // emoji表情作为图标
    hot: false,    // 是否热销
    sales: 0       // 初始销量
}
```

### 修改餐厅名称

在HTML文件中搜索"美味餐厅"并替换为你的餐厅名称。

## ⚠️ 注意事项

1. **数据持久化**: 
   - 当前使用LocalStorage存储数据，仅在当前浏览器有效
   - 如需多设备同步，需要接入后端服务器和数据库

2. **支付功能**: 
   - 当前为模拟支付，实际使用需接入微信支付API

3. **打印功能**: 
   - 建议使用Chrome浏览器打印
   - 打印前请预览效果

4. **移动端适配**: 
   - 已做响应式设计，支持手机和电脑访问
   - 建议在手机微信中测试扫码效果
